# TypeScript Data Structures

Common data structures, in TypeScript.

To start developing with TDD, run in watch mode:

```bash
npm run watch
```

To run coverage,

```bash
npm run cov
```

To update the docs,

```bash
npm run doc
```

To commit,

```bash
npm install -g commitizen
git cz
```

Using this [starter](https://github.com/bitjson/typescript-starter).
